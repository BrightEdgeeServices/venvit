PowerShell scripts to install customized Python Virtual Enviroments.


Installation
------------

.. code-block:: bash

    pip install Batch


Help
----
Help is supplied in the scripts itslef:

#. vn.ps1 -h
#. vi.ps1 -h
#. vr.ps1 -h
